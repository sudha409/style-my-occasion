# Style My Occasion 👗✨

Style My Occasion is a fashion‑based React website that helps users choose outfits for different occasions such as birthdays, weddings, office meetings, and parties.  
This project is created by Sudha, inspired by  passion for fashion design and creativity.

---

## 🌸 About the Project

Back in college, I designed outfits for a boutique, and  sketches were turned into real outfits.  
People loved my designs and gave amazing feedback.  
This project continues that dream  helping people find stylish outfits for special occasions.

---

## 🎯 Features

- 🎉 Birthday Outfit Suggestions  
- 💍 Wedding Outfit Ideas  
- 🏢 Office Meeting Styles  
- 🎊 Party Wear Recommendations  
- 🧥 Occasion Style Page  
- ⭐ Feedback Section  


---

## 🛠️ Technologies Used

- React JS  
- React Router  
- CSS (Custom Styling + Animations)  
- GitHub Pages (Deployment)
